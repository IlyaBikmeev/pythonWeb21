### Задачи на условные операторы:

#### 1. Треугольник
> На вход подаются три отрезка длины a, b, c. Определить, можно ли из этих отрезков построить треугольник.

##### Формат ввода:
> 3 вещественных числа a, b, c. (a, b, c > 0).

##### Формат вывода:
> Если из a, b, c можно построить треугольник, выведите "YES". В противном случае, выведите "NO".

##### Пример ввода
```bash
1.0 2.0 3.0 
```

##### Пример вывода
```bash
NO
```


#### 2. Треугольник с определением типа
> Написать программу для определения, можно ли из отрезков с длинами a, b, c построить треугольник.  Если можно, то выяснить тип треугольника (разносторонний, равнобедренный, равносторонний).

##### Формат ввода:
> В первой строке содержатся четыре вещественных числа: a, b, c. (a, b, c > 0)

##### Формат вывода:
> Если из данных отрезков можно составить треугольник, то выведите его тип: "разносторонний", "равнобедренный" или "равносторонний". В противном случае, выведите: "не треугольник"

##### Пример ввода 1
```bash
1.0 2.0 3.0 
```

##### Пример вывода 1
```bash
NO
```

##### Пример ввода 2
```bash
6.0 6.0 6.0 
```

##### Пример вывода 2
```bash
равносторонний
```


#### 3. Квадратное уравнение
> Квадратное уравнение имеет вид: Ax^2 + Bx + C = 0, где х - неизвестное. Найдите действительные корни данного уравнения.

##### Формат ввода:
> Если данное уравнение имеет 2 корня, выведите их через запятую. Если один корень, выведите его. Если корней не существует, выведите "NO SOLUTION"

##### Пример ввода 1
```bash
1 -2 3
```

##### Пример вывода 1
```bash
NO SOLUTION
```

##### Пример ввода 2
```bash
5 -7 -12
```

##### Пример вывода 2
```bash
1 2.4
```

##### Пример ввода 3
```bash
1 2 1
```

##### Пример вывода 3
```bash
-1
```

#### 4. Счастливый билет
> Четырехзначное число называется счастливым, если сумма первых двух его цифр равна сумме двух последних.
> Ваша программа должна читать четырехзначное число и определять, счастливое ли оно.

##### Формат ввода:
> В первой строке содержится целое четырехзначное число n (1000 <= n <= 9999).

##### Формат вывода:
> Выведите "YES", если число счастливое и "NO" в противном случае.

#### 5. Лежит ли точка внутри окружности
> Даны координаты центра окружности x0, y0 и ее радиус r. Также, дана точка (x, y). Определите, лежит ли данная точка внутри окружности

##### Формат ввода:
> В первой строке лежат вещественные числа x0, y0, r, (r > 0). 
> Во второй строке лежат два числа x, y.

##### Формат вывода:
> Выведите "YES", если данная точка лежит внутри окружности и "NO" иначе.

##### Пример ввода 1
```bash
0 0 5
6 3
```

##### Пример вывода 1
```bash
NO
```

##### Пример ввода 2
```bash
0 0 5
2 4
```

##### Пример вывода 2
```bash
YES
```

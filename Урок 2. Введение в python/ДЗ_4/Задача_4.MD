# Задача 4

Напишите программу, которая вычисляет значение выражения:
![Формула](https://ipc.susu.ru/2311.gif)

В первой строке содержатся вещественное число A (-3 < A < 3) и целое число n(1 <= n <= 20). Вывести результат с точностью 0.001
Пример ввода:
```bash
2.6574 20
```

Пример вывода:
```bash
224090313.412
```

#### Задача 5.

Дана последовательность из n целых чисел, каждое из которых находится в диапазоне от 0 до 100. 
Указать в данной последовательности количество вхождений числа, встречающегося чаще других.


#### Формат ввода:
В первой строке ввода содержится число n, в следующей – последовательность чисел, разделённых пробелом.

#### Формат вывода:
Количество вхождений числа, встречающегося чаще других.

Пример ввода:
```bash
5
1 3 2 4 1
```

Пример вывода:
```bash
2
```

Пояснение: в данном массиве число 1 встречается чаще всех остальных. Всего в массиве две единицы, поэтому ответ 2.

Указания к решению: Диапазон чисел у нас от 0 до 100, поэтому мы можем завести массив размера 101. 
a[i] в данном массиве будет символизировать количество вхождений числа i. Мы можем пройтись по входному массиву, встретили например число 4, увеличиваем a[4] на единицу.
После такого прохода у нас в массиве а будут лежать частоты вхождений каждого числа. Достаточно будет найти максимум в этом массиве и вывести его на экран.

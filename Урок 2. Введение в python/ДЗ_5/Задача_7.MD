## Задача 7.*

Даны n точек, заданные целой парой чисел (x;y). Среди всех возможных треугольников, которые можно составить из этих точек,
найти треугольник с минимальной площадью. Для вычисления площади треугольника можно использовать формулу Герона.

![Формула Герона](https://ds04.infourok.ru/uploads/ex/0840/0012f3ff-8730d71d/img3.jpg)

#### Формат ввода:
В первой строке ввода содержится число n. В следующих n строках - пары чисел x, y. 

#### Формат вывода:
Вывести наименьшую площадь с точностью 6 знаков после запятой.

#### Пример ввода:
```bash
4
3 2
5 2
5 4
3 7
```

#### Пример вывода:
```bash
2.000000
```

Указания к решению: точки вы можете хранить как массив кортежей. Для первого примера этот массив будет: [(3,2), (5, 2), (5, 4), (3, 7)]. 
Затем вы должны перебрать всевозможные тройки точек, следовательно здесь три вложенных цикла. Храним текущую минимальную площадь.
Для каждой тройки чисел проверить, можно ли вообще составить такой треугольник
(неравенство треугольника). Затем, если такой треугольник составить можно, посчитайте его площадь по формуле Герона: сначала посчитайте длину каждой стороны
(как расстояние между каждой парой точек), посчитайте полупериметр и подставьте все в формулу. Если посчитанная площадь меньше текущей минимальной, обновите ее.

## Задача 2.

Дано два множества, в каждом из которых n целых чисел. Найти элементы, которые входят в оба множества. Учитывать, что множества не содержат повторяющихся элементов.

#### Формат ввода:
В первой строке ввода содержится число n, во второй – элементы первого множества, разделённые пробелами, в третьей – элементы второго множества.

#### Формат вывода:
Найденные элементы, разделённые пробелом. Если таких элементов нет, вывести –1.

Пример ввода:
```bash
5
1 2 3 4 5
1 2 3 6 7
```

Пример вывода:
```bash
1 2 3
```

Указания к решению: Заведите два списка, считайте в них оба множества. Затем заведите пустой список для ответа. Затем для каждого элемента из первого списка проверьте, есть ли он во втором списке. Если есть - добавляем в ответ с помощью метода append. Затем останется лишь распечатать содержимое получившегося списка.

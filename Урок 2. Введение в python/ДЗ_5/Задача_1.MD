## Задача 1.

Дано n различных чисел. Найти среди них два числа, модуль разности которых имеет наименьшее значение. Вывести модуль разности.

#### Формат ввода:
В первой строке ввода содержится количество чисел n. 2 <= n <= 100. Во второй строке - числа, разделенные пробелом. Все числа неотрицательны.

#### Формат вывода:
В первой строке содержится минимальный модуль разности.

#### Пример ввода:
```bash
3
2 5 9
```

#### Пример вывода:
```bash
3
```

Пояснение: всевозможные модули: |2 - 5| = 3, |2 - 9| = 7, |5 - 9| = 4. Минимальный из них - 3.

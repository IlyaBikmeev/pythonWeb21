## Задача 3.

Дано два множества, в каждом из которых n целых чисел. Найти элементы первого множества, которые не входят во второе.
Учитывать, что множества не содержат повторяющихся элементов.

#### Формат ввода
В первой строке ввода содержится число n, во второй – элементы первого множества, разделённые пробелами, в третьей – элементы второго множества.

#### Формат вывода:
Найденные элементы, разделённые пробелом. Если таких элементов нет, вывести –1.

#### Пример ввода
```bash
5
1 2 3 4 5
1 2 3 6 7
```

#### Пример вывода
```bash
4 5
```

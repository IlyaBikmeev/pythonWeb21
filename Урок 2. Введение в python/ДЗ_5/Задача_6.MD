## Задача 6.
Для каждого элемента одномерного массива найти ближайший элемент в этом массиве (ближайшим элементом считается тот, разность с которым минимальна по модулю). 
Если таких элементов несколько, ближайшим считать элемент с наименьшим номером.


#### Формат ввода:
В первой строке ввода содержится число n - длина массива, во второй строке – элементы массива, разделённые пробелами, значения элементов в диапазоне от 0 до 1000.


#### Формат вывода:
Первое число – элемент, ближайший к первому элементу исходного массива, через пробел второе число – элемент, ближайший ко второму элементу исходного массива, и так далее.

#### Пример ввода:
```bash
4
1 9 3 7
```

#### Пример вывода:
```bash
3 7 1 9
```

Пояснение: 

Ближайший элемент к 1 = 3, т.к |1 - 3| = 2 - минимально возможная разность для единицы. 

Ближайший элемент к 9 = 7, т.к |7 - 9| = 2 - минимально возможная разность для девятки.

Ближайший элемент к 3 = 1, т.к |3 - 1| = 2 - минимально возможная разность для тройки.

Ближайший элемент к 7 = 9, т.к |9 - 7| = 2 - минимально возможная разность для семерки.

Итого, вывод: 3 7 1 9.

Указания к решению:

Пройдитесь по элементам входного массива. Для каждого из элементов сделайте проход в обратном порядке (for j in range(n, -1, -1)) 
и определите минимальную разницу для текущего элемента.
После прохода в обратном порядке для текущего элемента железобетонно будет найден ближайший элемент с наименьшим номером. 
Прямо в этом же цикле можете печатать этот найденный элемент через пробел.

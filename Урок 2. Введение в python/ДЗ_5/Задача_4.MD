## Задача 4.
Даны действительное число x и последовательность действительных чисел. В последовательности найти два члена, среднее арифметическое которых ближе всего к x.
Вывести среднее арифметическое. В случае нескольких вариантов ответа вывести минимальное из найденных средних арифметических значений, ближайших к x.

#### Формат ввода
В первой строке ввода содержится число x, во второй n. В третьей строке - n чисел - элементы последовательности.

#### Формат вывода
Одно число – среднее арифметическое с точностью до 6 знаков после запятой.

#### Пример ввода:
```bash
2.5
3
1 3 5
```

#### Пример вывода:
```bash
2.000000
```

Пояснение: "среднее арифметическое двух чисел ближе всего к х" означает, что модуль разности среднего арифметического и х минимальный. 
В примере ввода всевозможные значения средних арифметических:

a1 = (1 + 3) / 2 = 2

a2 = (1 + 5) / 2 = 3

a3 = (3 + 5) / 2 = 4

Рассмотрим разности модулей с числом х = 2.5:

|a1 - x| = |2 - 2.5| = 0.5

|a2 - x| = |3 - 2.5| = 0.5

|a3 - x| = |4 - 2.5| = 1.5

Минимально возможная разность равняется 0.5. Следовательно, у нас два кандидата: a1 = 2 и a2 = 3. 
По условию задачи нам нужно вывести минимальное из средних арифметических, если несколько кандидатов. Значит, ответ 2.

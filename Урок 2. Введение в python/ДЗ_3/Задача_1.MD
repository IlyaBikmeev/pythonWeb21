# Задача 1
#### Нечетные цифры

В первой строке содержится целое число N (1 < N < 10 ^ 9).  С помощью операций деления нацело и взятия остатка от деления определить, имеются ли в записи числа 
N нечетные цифры. Если имеются, то вывести "True", если нет – вывести "False"

Пример ввода:
```bash
2109045
```

Пример вывода:
```bash
True
```

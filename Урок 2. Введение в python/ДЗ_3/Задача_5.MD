# Задача 5

В первой строке содержится целое число N (1 <= N <= 10^9). Последовательность чисел Фибоначчи задается следующим образом: F[1] = 1. F[2] = 1. F[k] = F[k - 1] + F[k - 2] 
для k >= 3. Найти первое число Фибоначчи, большее N.

Пример ввода:
```bash
5144
```

Пример вывода:
```bash
6765
```
